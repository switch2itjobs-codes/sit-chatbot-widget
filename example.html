Simple Chatbot Widget Integration

This page demonstrates how to integrate the chatbot widget into your website.

Note: This example uses a placeholder webhook URL. In a real implementation, replace it with your actual API endpoint.

Implementation Code

<!-- Add this script tag before closing </body> tag --> <script src="chatbot-widget.js"></script> <script> ChatbotWidget.init({ webhookUrl: 'https://your-api-endpoint.com/webhook', agentName: 'Support Assistant', welcomeMessages: [ 'Hello! How can I help you today?' ] }); </script>
